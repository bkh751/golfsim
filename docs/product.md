# Product

## 1. 제품 정의
`golfsim`은 브라우저에서 동작하는 골프 시뮬레이터다.

## 2. 제품 목표
이 제품은 사용자가 직관적으로 플레이하고, 결과를 이해하고, 반복해서 개선할 수 있는 경험을 제공한다.

## 3. 제품 원칙
- 사용자는 항상 현재 상태와 결과를 이해할 수 있어야 한다.
- 결과 표현은 일관되어야 한다.
- 제품 복잡도는 사용자 가치보다 앞서지 않는다.
- 큰 구조 변경보다 작은 개선을 우선한다.

## 4. 판단 기준
새로운 기능이나 변경은 아래 기준으로 판단한다.

1. 사용자 경험을 더 명확하게 만드는가
2. 결과에 대한 신뢰를 높이는가
3. 복잡도 증가가 가치에 비해 과하지 않은가

## 5. 로드맵
- 브라우저 우선 확장과 차기 엔진 전환 준비 로드맵은 `docs/roadmap-browser-first-engine-transition.md`를 따른다.
