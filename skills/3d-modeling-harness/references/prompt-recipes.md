# Prompt Recipes

이 문서는 3D 비전문가 사용자와 짧게 합을 맞추기 위한 프롬프트 패턴만 담는다.

## Default response frame
- `상태`: 지금 무엇을 정리 중인지 한 줄
- `왜 지금 이 작업인지`: 현재 결정이 어떤 후속 작업을 잠그는지 한 줄
- `모델링 작업`: 이번 턴 산출물
- `리스크`: 아직 잠기지 않은 결정
- `사용자 확인`: 다음 한 가지 판단

## Example asks
- `#34 기준으로 Blender export 기본 규칙만 문서화해라.`
- `#45와 연결해서 필드 좌표계에서 먼저 잠가야 할 값만 정리해라.`
- `3D Modeling Team을 바로 추가하지 말고 승격 gate만 정의해라.`
- `내가 초보자라는 점을 전제로 glTF와 GLB 차이를 이번 결정에 필요한 수준으로만 설명해라.`

## Feedback loop
- 한 번에 하나의 선택지만 물어본다.
- 설명보다 결정에 필요한 차이만 비교한다.
- 사용자가 막힌 용어만 짧게 풀어준다.
- 다음 요청은 가능한 한 `예/아니오` 또는 `A/B` 수준으로 줄인다.
